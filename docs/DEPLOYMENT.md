# Deployment Guide

This project is deployment-ready for Streamlit-based hosting and Docker.

## 1) Local Run

```powershell
$env:PYTHONPATH='src'
c:/python314/python.exe run_pipeline.py
c:/python314/python.exe -m streamlit run streamlit_app.py
```

## 2) Streamlit Community Cloud

1. Push project to GitHub.
2. In Streamlit Cloud, select repo and branch.
3. Set app file path to `streamlit_app.py`.
4. Ensure `requirements.txt` is present.
5. Deploy.

Notes:
- If `artifacts/models/*.pkl` are not committed, run pipeline in CI/CD first and include model artifacts.
- The app expects reports and model files generated by `run_pipeline.py`.

## 3) Render / Heroku-like Procfile Deploy

`Procfile` included:

```text
web: streamlit run streamlit_app.py --server.port=$PORT --server.address=0.0.0.0
```

## 4) Docker Deploy

Build:

```bash
docker build -t churn-app .
```

Run:

```bash
docker run -p 8501:8501 churn-app
```

## 5) Recommended Pre-Deploy Checklist

- Run pipeline successfully.
- Confirm files exist:
  - `artifacts/models/best_model.pkl`
  - `reports/metrics_summary.csv`
  - `reports/model_recommendations.json`
  - `reports/selected_threshold.json`
- Start app and validate one prediction end-to-end.
